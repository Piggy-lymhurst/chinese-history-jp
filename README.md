# Chinese History in Japanese / 日本語で学ぶ中国史

An open learning website for studying Chinese history in Japanese.

This project presents Chinese history as a rope-like timeline: major eras are shown as large knots, and important events are shown as smaller knots. The goal is to make history easier to follow, compare, discuss, and improve together.

Live site:
https://piggy-lymhurst.github.io/chinese-history-jp/

## Features

- A Japanese-language timeline of Chinese history
- Rope and knot visual design for eras and events
- Key dates for important historical events
- Reader comments powered by GitHub Discussions and Giscus
- Simple static website using HTML, CSS, and JavaScript

## How To Contribute

Contributions are welcome. You can help by:

- Adding missing historical events
- Correcting dates or descriptions
- Improving Japanese explanations
- Suggesting better structure for the timeline
- Improving the design or mobile layout
- Opening discussions with questions or learning notes

For factual corrections, please include a source or short explanation when possible.

## Project Files

- `index.html` - main page content
- `style.css` - page design and rope timeline styling
- `script.js` - small JavaScript file
- `README.md` - project introduction

## Comments And Discussion

The website uses Giscus to connect page comments with GitHub Discussions.

The main historical content should stay factual and neutral. Reader comments, questions, and personal interpretations should stay in the comments/discussion area so that facts and opinions remain clearly separated.

# 日本語で学ぶ中国史 / Chinese History in Japanese

日本語で中国史を学ぶためのオープンな学習サイトです。

このプロジェクトでは、中国史を一本の縄のような年表として表現しています。大きな時代は大きな結び目、重要な出来事は小さな結び目として表示し、歴史の流れを見やすく、話し合いやすくすることを目指しています。

公開ページ:
https://piggy-lymhurst.github.io/chinese-history-jp/

## 特徴

- 日本語で読める中国史の年表
- 時代と事件を「縄」と「結び目」で表すデザイン
- 重要な歴史事件の年代表示
- GitHub Discussions と Giscus によるコメント機能
- HTML、CSS、JavaScript だけで作られたシンプルな静的サイト

## 参加方法

このプロジェクトへの参加を歓迎します。たとえば、次のような形で協力できます。

- 不足している歴史事件を追加する
- 年代や説明文を修正する
- 日本語の表現をより自然にする
- 年表の構成を提案する
- デザインやスマートフォン表示を改善する
- コメントや Discussions で質問、補足、学習メモを投稿する

事実関係を修正する場合は、可能であれば出典や簡単な理由も添えてください。

## ファイル構成

- `index.html` - メインページ
- `style.css` - ページデザインと縄の年表のスタイル
- `script.js` - 小さな JavaScript ファイル
- `README.md` - プロジェクト紹介

## コメントと議論

このサイトでは、Giscus を使ってページ上のコメントと GitHub Discussions をつなげています。

本文の歴史情報は、できるだけ客観的で中立的に保ちます。読者の感想、質問、解釈、補足はコメント欄や Discussions に分けることで、事実情報と意見を区別しながら学べるようにしています。
